# Game Pillars

- **Core fantasy:** self-aware industrial machine in human ruins.
- **Player verbs:** move, inspect, lift/shift, power/reactivate, record memories.
- **Tone:** melancholic, curious, quiet wonder.
- **Non-goals:** no combat, no crafting trees, no open-world streaming.
- **Target:** PC first, keyboard/mouse + controller, 60 FPS baseline.
